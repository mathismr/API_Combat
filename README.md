# API Combat

## Description

The Combat API for the [Gatcha API project](https://github.com/Maxe2005/GatchaApi).

> [!IMPORTANT]
> By lack of time and since this part of the project was a bonus — the skill choice AI and the cooldown computing code was generated by Gemini.

## Installation

```bash
$ git clone https://github.com/mathismr/API_Combat.git
$ docker-compose up --build 
```

Do not forget to create the `.env` file from the `.env.example` file and update it with the correct URL and port for the Monsters API.

## Usage

- Access the API at `http://localhost:8000`
- API Documentation at `http://localhost:8000/docs`

> [!WARNING]
> Do not re-use the code in this project for your production environments.
