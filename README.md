# API Combat

## Description

The Combat API for the [Gatcha API project](https://github.com/Maxe2005/GatchaApi).

## Installation

```bash
$ git clone https://github.com/mathismr/API_Combat.git
$ docker-compose up --build 
```

## Usage

### Access the API at `http://localhost:8000`

### API Documentation at `http://localhost:8000/docs`

Don't forget to create the `.env` file from the `.env.example` file and update it with the correct URL and port for the Monsters API.

> [!WARNING]
> Do not deploy this project for production environments.
